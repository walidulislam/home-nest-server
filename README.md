## Project Name: BDx Glory

BDx Glory is a modern e-commerce platform built for seamless online shopping experience. It allows users to browse products, add items to cart, and place orders with secure authentication and payment integration.

---

## Purpose
The purpose of this project is to build a full-stack e-commerce platform where users can easily explore products, manage their cart, and complete secure purchases online.

---

## Live URL
https://bdx-glory-website.netlify.app/

---

## Key Features
- Responsive and modern UI design  
- Product browsing and category-wise filtering  
- Shopping cart functionality  
- Secure user authentication (Login/Register)  
- Order management system  
- Smooth animations and user experience    

---

## Tech Stack / NPM Packages Used
- Next.js  
- Tailwind CSS  
- MongoDB  
- Firebase Authentication  
- Stripe  
- framer-motion  
- swiper
